Partner Badge Assets
====================

Replace these placeholder SVG files with the official badge images from each
partner portal. The site will display whatever file is present — PNG or SVG.
SVG is loaded first; if it fails, the code falls back to PNG automatically.

Files:
  microsoft-partner-badge.svg   — Microsoft AI Cloud Partner Program
  google-partner-badge.svg      — Google Partner badge
  quickbooks-proadvisor-badge.svg — QuickBooks Certified ProAdvisor

Download the official badges from:
  Microsoft  → https://partner.microsoft.com/ (Logo Builder)
  Google     → https://www.google.com/partners/ (Partner badge page)
  QuickBooks → https://quickbooks.intuit.com/accountants/proadvisor/ (badge download)

Accepted formats: .svg or .png (use the same base filename).
Recommended height: 48–60 px on-screen.
